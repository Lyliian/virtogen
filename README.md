# virtogen

# Requis

- PHP 8.3
- node > 20
- yarn
- composer

# Installation

- composer install
- yarn install
- faire une copie du .env en .env.local et remplir les données

# Stack utilisée
 - Symfony 7
 - (Node LTS)
 - Vue 3.5
 - Vite
 - Front Quasar

# Lancer le projet
- Back-end
  - sudo symfony server:start -d --port=80 
- Front-end
  - yarn dev / mode dev
  - yarn build / mode prod
